Pokédex

Description
Interactive Pokémon encyclopedia, browse the Pokémon world by type, name, or ID. Built on an external API with dynamic display and detailed views for each Pokémon.

Implementation Details
This project provided a practical introduction to working with external APIs. Using the public Pokémon API, I was able to fetch, process, and display data dynamically. In the process, I learned key concepts such as HTTP requests, asynchronous data handling, and error management in a frontend context.

Beschreibung
Interaktives Nachschlagewerk für Pokémon, durchsuche die Welt der Pokémon nach Typ, Name oder ID. Basierend auf einer externen API, mit dynamischer Anzeige und Detailansicht zu jedem Pokémon. Projektdetails.

Umsetzungsdetails
Dieses Projekt bot einen praktischen Einstieg in die Arbeit mit externen APIs. Über die öffentliche Pokémon-API konnte ich Daten dynamisch abrufen, verarbeiten und darstellen. Dabei lernte ich zentrale Konzepte wie HTTP-Requests, asynchrone Verarbeitung und Fehlerbehandlung im Frontend-Kontext kennen.
