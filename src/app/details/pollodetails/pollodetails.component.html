Pollo Loco

Description
Jump, run and throw game based on object-oriented approach. Help Pepe to find coins and tabasco salsa to fight against the crazy hen.

Implementation Details
Building this game was an exciting addition to my previous experience in web development. Unlike traditional website projects, it required me to deal with real-time interactions, game logic, state management, and frequent visual feedback. Through this, I was able to broaden my technical understanding and explore new aspects of interactive application development.

Beschreibung
